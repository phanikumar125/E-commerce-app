# ShopSphere E-Commerce App

A complete React e-commerce frontend built with Vite, Redux Toolkit, React Router, and Tailwind CSS.

## Features

- Product listing with search, category filter, sort, and price range
- Product details page
- Cart with quantity management
- Wishlist
- Checkout flow with address, delivery, payment, and order summary
- Mock authentication pages
- Orders page
- Profile/settings page
- Dark/light theme
- Responsive UI

## Run Locally

```bash
npm install --legacy-peer-deps
npm run dev
```

Open the local URL shown by Vite.
